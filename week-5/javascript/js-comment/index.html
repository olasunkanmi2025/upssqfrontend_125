Javascript Comments

As with html and css, it is possible to write comments in your javascript code that will be ignored by the browser. comment provide instruction to your fellow developers on how the code works (and you if you come back to your code after six month and can't rememeber what you did). comment are very useful, and you should use them often, particularly for larger aplication. There are two types of comment:

  • A "single line comment" is written after a doubled forward slash("//,e.g")

    js 
                        
                        //I am a comment

  • A "multi line comment" is written between the string "/*" "and"*/"example"

    js 
                        
                        /*I am a comment */
