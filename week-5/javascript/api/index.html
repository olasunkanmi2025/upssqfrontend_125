Application programming interface(APIs)

APIs are ready made set of code building blocks that allows a developer to implement that will otherwise be hard or impossible to implement they do the same for programming that ready made furniture kit do for home building. it is much easier to take ready cut panel and them screw them together

Browser API

browser api are built into your web browser and are able to expose data from the surronding computer environment or do useful complex things for example

  • The DOM(document object model)

    the document object model allows you to manipulate html and css by creating, removing and changing html by dynamically applying new style to your page among other things. Anytime you see a popup appear on your page or some new content displays that is DOM in action like the exmple we did above

  • The geo-location api

    The geo-location api retrieve geo-graphical information. this is how google maps is able to find your location and plot it on a map

  • The canvas and webGl APIs

    The canvas and webGL APIs allows you to create animated 2D and 3D graphics

  • Audio and Video APIs

    Example: Html Media elements and webRTC allows you to do many interesting with media such as play audio and video and display it on someone else computer

Third party APIs

Third party APIs are not built into the browser by default and you generally have to grab their code and information from somewhere on the web for example:

  • Bluesky Api

    allows you to do things like displaying your latest post on your website

  • Google Maps API and OpenStreetMap API

    Google Maps API and OpenStreetMap API allows you to embed custom map into your website and other such functionality
