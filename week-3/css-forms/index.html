Html Form

An html form is use to collect user input. The user input is most often sent to the server for processing

The form element

The form element is used to create an html form for the user input:

The form element is a container for different type of input element such as text fills, check buttons, radio buttons and submit button

The input element

The html input element is the most use element

Type Description
Input type text It displays the single line text fill
Input type radio It defines a radio button input fill that allows selections of one option from a set
Check box Define a check box page that allows selection of multiple option from a set
Input Type Number This define an input fill from entering a number
Input Type Email Defines an input fill for entering an email address
Input type password Defines a password input fill
Input type file This allow the upload of a file from ur computer
