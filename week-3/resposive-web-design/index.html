resposive-web-design(RWD)

resposive-web-design(RWD) is a design approach that ensure a website looks and functions well across a wide range of devices and screen sizes. - From mobile phones to tablet to desktop computers-Without requiring separate versions of the site

Key concepts of resposive web design:

  1. Fluid grids:

    Layout are built using relative unit like percentages instead of fixed unit like pixel allowing element to resize based on the screen

  2. Flexible images:

    Images scale within their containing elements so they overflow or distort on different screen sizes

  3. Media query:

    Css rules that apply styles based on the devices characteristics such as screen weight, height, resolution or orientation

    Example:
    
                        @media (max-width: 768px) {
                                .menu {
                                       display: none;
                                      }
                                 }

    Mobile-first approach:

    Design start with the smallest screen size and scales up, ensuring a good user experience on mobile design first

Grid options in resposive design

  • Css grid layout:

    Allows for complex layouts with rows and columns making it easier to create resposive design

  • Flexbox:

    Provide a one dimensional layout model that is great for aligning items in a rows or columns making it easier to create flexible layouts

  • Frameworks:

    Framework like bootstrap, bulma, w3css, tailwind css and foundation provide prebuilt resposive grid systems that can be easily integrayed into projects.
