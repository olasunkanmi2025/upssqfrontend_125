Generic font family

In css, there are five generic font family:


  1. font family-seref: seref font has a small stroke at the edge of each letter they create a sense of formality and elegant
  2. Sans-seref: This font has a clean line, (no small stroke attached) They create a modern amd minimalistic look
  3. Mono space: This font have all their letters with the same fixed width. They create a mechanical look
  4. Cursive font: This font imitate human handwritings
  5. Fantasy font: This are decorative/playful fonts

All the different font name belongs to one of the generic family in design, font family are referred to as type phase
