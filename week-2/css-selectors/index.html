overflow property description

The overflow property specify whether to click the content or add scroll bars when the content of an element is too big to fit the specify area

Overflow property has the following values visible, hidden, scroll, auto

Note:The overflow property only works for block element that has specify height. In oxxlion is the name of an operating system on a mac computer and scroll bars are hidden by default and only show when been used even if the overflow is set on scroll
