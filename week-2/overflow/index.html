overflow property description

The overflow property specify whether to click the content or too add scroll bars when the content of an element is to big to fit the specify area

The overflow property has the following properties: visible, scroll, hidden, auto

Note: The overflow property only works for bloch elements with specify height. In oxxlions is the name of an operating system on a mac computer and scroll bars are hidden by default on this operating system and only shows when been used even if the overflow is set on scroll

overflow-x overflow-y

The overflow-xand overflow-y properties specifies whether to change the overflow of content just vertically or horizontally or both

overflow-xspecifies what to do with the left/right edges of the contents.

overflow-x specifies what to do with the top/buttom edges of the contents.



You can use the overflow property when you want to have better control of the layout. The overflow property specify what happens if the contents overflow an element box
